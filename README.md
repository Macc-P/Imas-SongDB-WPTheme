# Imas-SongDB-WPTheme
アイドルマスターの楽曲DBで使っているWordPress用テーマです。  
最新版のミリオンライブ用DBのデータをアップしています。  
ライセンスや書き直しの関係もあり、一部のコアファイルのみの公開としています。  
IssueやPullRequestはどんどんしてもらって大丈夫です。
